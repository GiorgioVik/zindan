## Zindan 1.5.3 (235)

Field-tested on Samsung Galaxy S24 Ultra and Tab S9 FE+ (June 2026).

**APK:** `Zindan-1.5.3-(235)-debug.apk` (debug-signed)

### Highlights

- Work profile list refreshes after RuStore / store install while Zindan is open — no tab switch
- VPN auto-freeze for store-installed apps; Samsung poll loop fixed (build 231)
- Menu **Install APK into Zindan** blocked while VPN is active (build 229)
- Auto-freeze snowflake assigned to apps installed in the work profile from stores

### Install

Download the APK below. Update over an existing install in **both** profiles (personal first, then work) — no need to recreate the work profile when `versionCode` increases.

See [CHANGELOG.md](https://github.com/GiorgioVik/zindan/blob/main/CHANGELOG.md) and [USER_GUIDE.md](https://github.com/GiorgioVik/zindan/blob/main/USER_GUIDE.md).
