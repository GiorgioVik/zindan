## Zindan 1.5.4 (243)

**APK:** `Zindan-1.5.4-(243)-debug.apk` (debug-signed)

### Highlights

- Anti Spy VPN watcher hardening for long-running use on Samsung devices
- Poll self-heal when internal VPN state desyncs from reality
- 30-minute watchdog + background FGS restart retry
- **Settings → Services → Anti Spy VPN monitor** — status and one-tap restart
- User guide warning: never clear Zindan app data in either profile

Includes all [1.5.4 (242)](https://github.com/GiorgioVik/zindan/releases/tag/v1.5.4-242) freeze-engine and VPN coordinator improvements.

### Install

Download the APK below. Update over an existing install in **both** profiles (personal first, then work) — no need to recreate the work profile when `versionCode` increases.

See [CHANGELOG.md](https://github.com/GiorgioVik/zindan/blob/main/CHANGELOG.md) and [USER_GUIDE.md](https://github.com/GiorgioVik/zindan/blob/main/USER_GUIDE.md).
